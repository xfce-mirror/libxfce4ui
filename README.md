libxfce4ui
====================

The libxfce4ui library is used to share commonly used Xfce widgets among the Xfce applications.

### Installation

From source: 

    % cd libxfce4ui
    % ./autogen.sh
    % make
    % make install

From release tarball:

    % tar xf libxfce4ui-<version>.tar.bz2
    % cd libxfce4ui-<version>
    % ./configure
    % make
    % make install
